# Contributing

## Development
- Use Node.js 20+.
- Run `npm install` from repo root.
- Keep API keys out of git (use `.env` files).

## Code Style
- Prefer clear, readable code over cleverness.
- Keep UI logic in app packages and shared types in `packages/shared`.

## Running
```bash
npm run dev:server
npm run dev:web
npm run dev:mobile
```

## Pull Requests
- Include a short description of changes.
- Mention any new env vars.
